# 🧹 Data Sweeper

> A powerful data cleaning and transformation web app built with Python and Streamlit. Upload messy CSV or Excel files, clean them with one click, visualize your data, and download the results instantly.

🔗 **Live Demo:** [ayeshashahzaib-data-sweeper-app-tdjab7.streamlit.app](https://ayeshashahzaib-data-sweeper-app-tdjab7.streamlit.app/)

---

## ✨ Features

- 📂 **File upload support** — accepts CSV and Excel (.xlsx) files
- 🧽 **Automated data cleaning** — removes duplicates and fills missing values
- 🔄 **Column-level control** — select which columns to clean or keep
- 📊 **Built-in data visualization** — generate charts directly from your dataset
- 💾 **Export cleaned data** — download results as CSV or Excel
- 🔀 **Format conversion** — convert between CSV and Excel formats in one click

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Python | Core logic |
| Streamlit | Interactive web UI |
| Pandas | Data manipulation & cleaning |
| OpenPyXL | Excel file read/write support |

---

## 🧠 How It Works

1. Upload a CSV or Excel file through the web interface
2. Preview the raw data in an interactive table
3. Select cleaning operations (drop duplicates, fill missing values, etc.)
4. Choose specific columns to include or exclude
5. Visualize the cleaned data with built-in charts
6. Download the output in your preferred format

---

## 🚀 Run Locally

```bash
git clone https://github.com/AyeshaShahzaib/Data_Sweeper.git
cd Data_Sweeper
pip install -r requirements.txt
streamlit run app.py
```
